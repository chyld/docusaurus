# Xplore

| Command | Description                 |
| ------- | --------------------------- |
| `x`     | Change to project directory |
| `xc`    | Create a new project        |
| `xl`    | Short listing               |
| `xll`   | Long listing                |
| `xf`    | Fuzzy directory find        |
| `xs`    | Move from Temp to Save      |
| `xp`    | Purge temp folders          |
